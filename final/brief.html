Final Project Brief

Project Summary

I will update the Every Picture project's visual design and UX/UI.

Project Description

The Every Picture project features a gallery of photos taken from aquariums. Each image has a short description of the particular animal, discussing their geographic location, dimensions, and distinct physical characteristics.

Regarding the project's visual design, I will fix the transition between the first to second pages. It is currently an instantaneous change, and I want to add a zoom transition for the first page that will transition into the next. I will also revise the spacing issues with the images, specifically at the end. The last photo's margin does not appear, and provides no spacing between the edge of the page with the photo.

As for the UX/UI aspect, an arrow on the front of the second page will be added to help indicate to the user that you can scroll horizontally to view the photos, as a horizontal scroll is not intuitive for the user. Text will appear after the scroll to show you can click on the photo to learn more. At the right end of the page, I will include a button that lets the user scroll and jump back to the front beginning of the page. This prevents them from scrolling excessively to go to the beginning.
